Interface JwtHeader

Represents the header of a JSON Web Token (JWT).

The JwtHeader interface defines the structure of the JWT header, which contains metadata about the token, including the signing algorithm and key identifier. Additional properties may be present depending on specific use cases and extensions.

JwtHeader

interface JwtHeader {
    alg: Algorithm;
    kid?: string;
    typ?: string;
    x5c?: string[];
    x5t?: string;
    x5t_S256?: string;
    x5u?: string;
    [key: string]: any;
}
