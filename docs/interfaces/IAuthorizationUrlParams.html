Interface IAuthorizationUrlParams

Represents the parameters required to construct an authorization URL.

The IAuthorizationUrlParams interface provides a structured way to specify parameters for initiating an OpenID Connect or OAuth 2.0 authorization flow.

IAuthorizationUrlParams

    clientId: string;

    display?: string;

    prompt?: string;
    redirectUri: string;

}
