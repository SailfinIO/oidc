Interface Jwk

Represents a JSON Web Key (JWK).

The Jwk interface defines the structure of a single JWK, which is a JSON data structure that represents a cryptographic key. JWKs are used in various web security standards, including JWT (JSON Web Tokens) and OAuth 2.0.
