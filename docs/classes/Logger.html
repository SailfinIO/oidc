setLogLevel

  • setLogLevel(level: LogLevel): void

  • Sets the log level for the logger, defining the minimum severity of messages to log.

    Parameters

    • level: LogLevel

      The log level to apply (e.g., LogLevel.ERROR, LogLevel.DEBUG).

    Returns void

    Remarks

    Log levels are hierarchical, with DEBUG being the most verbose and ERROR the least. Messages at or above the specified level will be logged. The default log level is INFO.

    Example

    logger.setLogLevel(LogLevel.DEBUG);
