Class Jwks

Represents a client for managing and retrieving JSON Web Keys (JWKs) from a JWKS URI.

The Jwks class handles fetching, caching, and validating JWKS responses, and provides methods to retrieve a JWK by its Key ID (kid).

getKey

  • getKey(kid: string): Promise<Jwk>

  • Retrieves a JSON Web Key (JWK) by its Key ID (kid).

    This method first attempts to retrieve the JWK from the cache. If the key is not found or the cache is empty, it refreshes the cache by fetching the JWKS from the URI.

    Parameters

    • kid: string

      The Key ID to search for.

    Returns Promise<Jwk>

    The corresponding JWK.

    Throws

    If the kid is invalid or the key is not found.

    Example

    const jwk = await Jwks.getKey('myKid');
    console.log(jwk);
